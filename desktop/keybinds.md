# Maestro Ubuntu Keybinds Reference

Standard GNOME/Ubuntu shortcuts for Ubuntu 24.04 LTS.  
Source: [GNOME Help — Keyboard Shortcuts](https://help.gnome.org/users/gnome-help/stable/keyboard-shortcuts-set.html.en)

## System
| Shortcut | Action |
|----------|--------|
| `Super` | Show Activities overview / app search |
| `Super + A` | Show all installed applications |
| `Super + S` | Open system menu (top-right panel) |
| `Super + V` | Show notification list |
| `Super + N` | Focus active notification |
| `Super + L` | Lock screen |
| `Alt + F2` | Run command prompt |
| `Ctrl + Alt + Delete` | Power off dialog |

## Windows
| Shortcut | Action |
|----------|--------|
| `Super + ←` | Split window to left half |
| `Super + →` | Split window to right half |
| `Super + ↑` | Maximize window |
| `Super + ↓` | Restore/unmaximize window |
| `Super + H` | Hide (minimize) window |
| `Alt + F4` | Close window |
| `Alt + F10` | Toggle maximize |
| `Alt + Space` | Window menu (move, resize, etc.) |

## Workspaces
| Shortcut | Action |
|----------|--------|
| `Super + Page Up` | Switch to workspace on the left |
| `Super + Page Down` | Switch to workspace on the right |
| `Shift + Super + Page Up` | Move window to workspace on the left |
| `Shift + Super + Page Down` | Move window to workspace on the right |

## Applications
| Shortcut | Action |
|----------|--------|
| `Ctrl + Alt + T` | Open GNOME Terminal |
| `Super + E` | Open Files (Nautilus) — Ubuntu default |
| `Super + Tab` | Switch applications |
| `Alt + Tab` | Switch windows |
| `Alt + F6` | Switch windows of same application |

## Terminal
| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + T` | New terminal tab |
| `Ctrl + Shift + N` | New terminal window |
| `Ctrl + Shift + W` | Close tab |
| `Ctrl + Shift + C` | Copy |
| `Ctrl + Shift + V` | Paste |
| `Ctrl + D` | Close shell |

## Screenshot
| Shortcut | Action |
|----------|--------|
| `Print Screen` | Launch screenshot tool (select mode in UI) |
| `Alt + Print Screen` | Save screenshot of active window |
| `Shift + Print Screen` | Save screenshot of full screen |
| `Ctrl + Shift + Alt + R` | Record a short screencast |
