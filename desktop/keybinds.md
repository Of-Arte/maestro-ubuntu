# Maestro Ubuntu Keybinds Reference

These are standard GNOME/Ubuntu shortcuts customized to optimize your workflow.

## General
- `Super + Space`: Open application launcher
- `Super + Arrow Keys`: Snap window to left, right, top, or bottom
- `Super + Page UP/Down`: Switch workspace

## Applications
- `Ctrl + Alt + T`: Open GNOME Terminal
- `Super + B`: Open Web Browser (if configured)
- `Super + E`: Open File Manager
- `Super + L`: Lock Screen

## Workspace Management
- `Alt + Tab`: Switch between open applications
- `Super + D`: Show desktop (minimize all)

## Screenshot
- `Print Screen`: Take screenshot / open screenshot tool
